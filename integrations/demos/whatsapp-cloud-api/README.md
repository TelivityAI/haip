# WhatsApp Cloud API — demo

```bash
./integrations/demos/run.sh whatsapp-cloud-api
```

1. Turns ON the Integrations catalog toggle for `whatsapp-cloud-api`.
2. Without Cloud credentials, WhatsApp falls through to Twilio or console.

See [GO_LIVE.md](./GO_LIVE.md). Docs: [docs/integrations/whatsapp-cloud.md](../../docs/integrations/whatsapp-cloud.md)
