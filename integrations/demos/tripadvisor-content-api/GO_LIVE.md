# Go live — TripAdvisor Content API

## Demo
```bash
./integrations/demos/run.sh tripadvisor-content-api
```
Without keys the reviews module uses the **console** source.

## Live in a jiffy
1. Put API keys from [`demo.env.example`](./demo.env.example) in `.env`, restart API.
2. Pull reviews via the reviews API / dashboard and confirm vendor data.
